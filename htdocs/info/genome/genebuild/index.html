An overview of genome annotation

Ensembl Rapid Release produces annotation of new genomes from a variety of sources. These can include:

  • Ensembl genebuild
  • Ensembl genebuild for non-vertebrate species
  • Ensembl BRAKER2 annotation
  • Imported annotation from community resources
  • Curated annotation

The type of annotation produced is indicated on each species' home page.
